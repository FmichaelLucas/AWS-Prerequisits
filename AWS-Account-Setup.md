## Creating My AWS Account
In this step, I'm going to... Sign up for a new AWS account, verify my email address, and create a strong root user password.

### Understanding the root user
I learned that a root user is... a the main account owner that has full access to everything in your AWS account. A strong password is important in order for your account to be safe against potential data breaches.

### Setting Up Billing and Payment
In this step, I'm going to... set up my billings, by doing so, I will choose my account plan, enter my contact information, and finally add a payment method.

### Why AWS requires a payment method
I learned that AWS requires a payment method because... They want to make sure that I am indeed a real person, and I chose to go past my free limits, they will then have a way to charge me for it.
